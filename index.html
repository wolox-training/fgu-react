Developer Tools

Front-End Tools

ReactJS Release: 2013 React is a declarative, component-based JavaScript library for building user interfaces.
AngularJS Release: 2010 AngularJS (commonly referred to as AngularJS 1.X) is a Javascript framework for building SPAs with an MVC arquitecture.
VueJS Release: 2014 VueJS (pronounced /vju:/, like view) is an open-source progressive Javascript framework for building user interfaces.
Elm Release: 2016 Elm is a domain-specific functional programming lenguaje for declaratively creating web browser-based graphical user interfaces.
Polymer Release: 2015 Polymer is an open-source JavaScript library for building web applications using Web Components.

Back-End Tools

Ruby on Rails Release: 2005 Ruby on Rails, or Rails, is a server-side web application framework written in Ruby. Rails is an MVC framework, providing default structures for a database, a web service, and web pages.
NodeJS Release: 2009 NodeJS is a Javascript runtime built on Chrome's JavaScript engine. Uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.
ASP.NET Core Release: 2016 ASP.NET Core is the next generation of ASP.NET. It is a modular framework that runs both the full .NET framework, on Windows, and the cross-platform .NET Core.
Django Release: 2005 Django (/dʒængʊ/ JANG-goh) is a free open-source web framework, written in Python, wich follows the Model-View-Template (MVT) arquitectural pattern.
